Glass Spades Real Estate is a young real estate agency in Kenya. It is an organization that understood the necessity for a modern website using modern technologies. They wanted a website that would compete well and give the young company a fighting chance in a competitive market space under a tough economy.

User experience was the main focus both for them and for me. We understood that majority of our users are on mobile phone. We therefore created a website that was both responsive and accessible to all our users. We were going for a feel that was both clean and elegant in order to bring out the best of the houses that they were marketing.

The MEA Tax Project is a project that I undertook for the US mission employees. According to the US laws, employees file their own taxes and pay their own government while in Kenya it is the opposite. Employers withhold their employees taxes and file them with the Kenya Revenue Authority. This puts a lot of these employees under undue stress as many are not familiar with the relevant tax laws to start doing so on their own.

The employees came together and formed an association that assists them to create the necessary tax file, P9 and provides guidance on how to go about it. The number of employees in the association is over 400. The due date for the P9 forms is on June 30. For two years, the accountants were collecting the necessary financial information and creating the P9 forms for each employee. It was a strenous task.

The program compresses what was 6 months of labor into 5 minutes. All of which is error free. It has allowed the accountants to focus on other tasks and the employees to file their taxes on time. All the program needs is the employee information such as their name, tax pin, and salary information. The program them computes the taxes due per employee. It then creates the P9 form which is an excel form converted into a PDF and then shared with the employees.

The tech stack used in the project is Python which has a very good excel package, Openpyxl. The file conversion is done through Linux to automate the file conversion.
